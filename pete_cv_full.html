Featured Project
Deep Learning Breast Cancer Classification — CM3070 Final Project, University of London (2026)
github.com/peter-kautwima/DeepLearning-BreastCancer-Classification
  • Built an end-to-end deep learning pipeline for malignant/benign mammogram classification on the CBIS-DDSM clinical dataset (~3,500 biopsy-verified ROI patches).
  • Four-stage iterative pipeline: Baseline CNN → Frozen VGG16 → Regularisation (Dropout + L2) → Fine-tuning. Best model: AUC 0.7678, sensitivity 0.7055 on 714 held-out images — 64% relative sensitivity gain over baseline, exceeding mean individual radiologist AUC (McKinney et al. 2020).
  • Implemented Grad-CAM explainability via a custom GradientTape workaround for a Keras 3 graph disconnection issue — heatmaps for all four prediction categories (TP, TN, FP, FN).
  • Fully reproducible on a public dataset with stratified splits, class-imbalance weighting, and training-only augmentation.
